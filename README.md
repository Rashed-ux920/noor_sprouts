# 🌱 Noor Sprouts – Landing Page

A colorful and kid-friendly landing page for **Noor Sprouts**, an educational platform that helps children learn Quran, Arabic, and Islamic values in a fun and engaging way.

---

## 🎯 Project Overview

This project is a **static landing page** built using:

* HTML
* CSS
* JavaScript

It is designed for:

* 👶 Kids (ages 4–10)
* 👨‍👩‍👧 Parents looking for safe and interactive learning

---

## 🎨 Design Concept

* Bright and playful colors (Yellow, Green, Blue)
* Rounded UI elements
* Soft shadows and gradients
* Friendly and engaging layout

---

## ⚙️ Tech Stack

* **HTML5**
* **CSS3**
* **Vanilla JavaScript**
* Fully responsive (mobile-first design)

---
